Web Development

Build the internet — frontend, backend & full-stack

💻 Device Requirements
Processor
Intel i5 / Ryzen 5+
RAM
8 GB minimum
Storage
256 GB SSD
OS
Windows / macOS / Linux
🛠️ Tools & Software

  • Databases — MongoDB, PostgreSQL, Firebase
  • Deployment — Vercel, Netlify, AWS, Docker basics

Frontend Developer Backend Developer Full-Stack Developer UI Engineer DevOps Engineer

Android Development

Build apps for 3 billion+ Android devices worldwide

💻 Device Requirements
Processor
Intel i5 / Ryzen 5+ (multi-core)
RAM
16 GB recommended
Storage
512 GB SSD (emulators need space)
Extra
Android phone for testing
🛠️ Tools & Software
  • Android Studio — Official IDE with emulator & debugger
  • Kotlin — Modern primary language for Android
  • Firebase — Auth, database, analytics, push notifications
  • Git & GitHub — Version control
  • Figma — UI/UX design for mobile screens
  • ADB — Android Debug Bridge for device testing
📚 Learning Path
  • Java or Kotlin fundamentals — OOP, collections, coroutines
  • Android Studio setup — Project structure, Gradle
  • UI Design — XML Layouts or Jetpack Compose
  • Navigation & Architecture — MVVM, ViewModel, LiveData
  • APIs & Networking — Retrofit, OkHttp, REST APIs
  • Local Storage — Room Database, SharedPreferences
  • Firebase Integration — Auth, Firestore, Cloud Messaging
  • Publishing — Google Play Store deployment
🎯 Career Outcomes
Android Developer Mobile Engineer Kotlin Developer App Architect
💰 ₹6 LPA – ₹30 LPA (fresher to mid-level)

AI / Machine Learning

Teach machines to think, learn, and create

💻 Device Requirements
Processor
Intel i7 / Ryzen 7+ recommended
RAM
16 GB minimum (32 GB ideal)
GPU
NVIDIA GPU with CUDA (or use Colab)
OS
Linux preferred, macOS / Windows OK
🛠️ Tools & Software
  • Python 3 — Primary language for ML/AI
  • Jupyter Notebook / Google Colab — Interactive experimentation
  • TensorFlow & PyTorch — Deep learning frameworks
  • scikit-learn — Classical ML algorithms
  • Pandas & NumPy — Data manipulation & math
  • Hugging Face — Pre-trained models & NLP pipelines
📚 Learning Path
  • Python Programming — Data types, OOP, file I/O
  • Mathematics — Linear algebra, calculus, probability & statistics
  • Data Analysis — Pandas, NumPy, Matplotlib, EDA
  • Machine Learning — Supervised, unsupervised, ensemble methods
  • Deep Learning — Neural networks, CNNs, RNNs, Transformers
  • Specialization — NLP, Computer Vision, or Reinforcement Learning
  • MLOps — Model deployment, monitoring, CI/CD for ML
🎯 Career Outcomes
ML Engineer Data Scientist AI Researcher NLP Engineer Computer Vision Engineer
💰 ₹8 LPA – ₹40 LPA (fresher to mid-level)

Cybersecurity

Defend systems, hunt threats, and break to protect

💻 Device Requirements
Processor
Intel i5 / Ryzen 5+ (VMs needed)
RAM
16 GB recommended (multiple VMs)
Storage
512 GB SSD (Kali + VMs)
OS
Kali Linux + Windows dual-boot
🛠️ Tools & Software
  • Kali Linux — Penetration testing distribution
  • Burp Suite — Web application security testing
  • Wireshark — Network packet analysis
  • Metasploit — Exploitation framework
  • Nmap — Network scanning & enumeration
  • VirtualBox / VMware — Running isolated lab environments
📚 Learning Path
  • Networking Fundamentals — OSI model, TCP/IP, DNS, HTTP
  • Linux Administration — Command line, users, file permissions
  • Web Security — OWASP Top 10, XSS, SQL injection, CSRF
  • Cryptography — Encryption, hashing, PKI, SSL/TLS
  • Penetration Testing — Recon, scanning, exploitation, reporting
  • Digital Forensics — Disk imaging, memory analysis, log review
  • Bug Bounty — HackerOne, Bugcrowd, real-world hunting
🎯 Career Outcomes
Security Analyst Penetration Tester Security Engineer SOC Analyst Bug Bounty Hunter
💰 ₹6 LPA – ₹35 LPA (fresher to mid-level)
